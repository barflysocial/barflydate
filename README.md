## New in v81 — No Hero Graphic Option
Built from v80.

### No Hero Graphic
Add/Edit Event now includes a clean **No Hero Graphic** option.

You can choose it from the Hero Template Gallery:

`No Hero Graphic → Use clean text-only event card`

When selected:
- Hero Image URL is cleared
- Hero graphic is turned off for that event
- The public event card does not show a blank image box
- The event detail page does not show a hero image
- The preview shows a clean text-only layout

### Advanced setting
Advanced Event Details now includes:

`Use hero graphic for this event`

Turn it off when you want a simple event card with no hero image.

### Packaged Workday Blackout Bingo hero
The uploaded Workday Blackout Bingo graphic was packaged into the app at:

`/heroes/barfly-social/workday-blackout-bingo.png`

You can paste that path into a Venue Hero Template or Event Hero Image URL.
