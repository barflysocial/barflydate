# Barfly Social v125 Changes

Base used: `BARFLYDATE_v124_20260505_1349_4ICON_HOME_RENDER_READY(3).zip`

## Event social promo URLs

Added host event fields for:

- YouTube Short URL
- TikTok URL
- Instagram Reel / Post URL

Public event pages and event cards now show those buttons only when the saved event has a URL. Blank URL fields do not render a public button.

## About Us advertising page

Added `Advertise With Us` under About Us.

The icon opens a description card explaining advertising opportunities, including:

- Event sponsorships
- In-app promo placements
- Barfly TV ads
- Social wall features
- Prize partnerships
- Custom activations

An `Email to Apply` button uses the Advertising Email setting when present, then falls back to Business Email.

## Host settings

Added settings for:

- Advertising Email
- Advertise With Us description text
